# Criação de um portfolio de investimentos usando o pacote Tidyquant e o modelo CAPM

<p>Este projeto tem por objetivo construir um portfólio de ações da bolsa brasileira, utilizando como base teórica o CAPM - Capital Asset Pricing Model.
Para desenvolver esse exercício, estou usando o pacote Tidyquant, disponível em:
https://cran.r-project.org/web/packages/tidyquant/vignettes/TQ05-performance-analysis-with-tidyquant.html
</p>
<p>Observação importante. Não se trata de recomendação de investimentos e tão pouco tem por objetivo orientar os usuários que chegaram a este projeto a investir nessa carteira de ações. Lembre-se que investir em ações requer tempo e dedicação aos estudos, além do acompanhamento diário do mercado.</p>
<p>O portfólio criado foi obtido a partir de uma lista de 20 ações que tiveram retorno médio positivo nos últimos 6 meses. A simulação da carteira levou em consideração um valor hipotético investido de R$ 100 mil, distribuído de forma arbitrária e aleatória entre os papéis escolhidos, para um horizonte de tempo de um mês, especificamente o mês de outubro de 2021.</p>
<p>Não recomendo utilizar este portfólio para distribuir seus investimentos. Para tal, faça uma consulta à um economista!</p>
